Getting started
To get started using Quantext, simply sign up a new account or log in with a Google account (currently we only support Google sign in; more options coming soon!) Quantext is free to use for basic analyses (see BETA SIGNUP for more information on upgrading your Quantext account to the BETA).
Control panel
After logging in to Quantext, you will be taken to your control panel. From here, you can manage your files, and run data analyses.
Getting your data into Quantext
Uploading a new file - make sure your data is in the correct format for Quantext. Refer first to the Quantext data file format guide for full details of the required format, limitations and caveats.
Click the 'Files' tab in the menu bar, or the
icon (Upload) on the main dashboard.
The File Manager panel will open. Click upload new file and browse to locate the file you want to upload. Quantext files are .xlsx files and must be in the correct format. Other file types can be uploaded for use as reference files; currently only PDF files are supported. When you have selected your file, click upload.
Managing your files
All the files you have uploaded are listed in the File Manager, ordered by most recently uploaded. Select one or more files for analysis by selecting the checkbox next to each filename. You can filter files by type by selecting the appropriate checkbox at the top of the file list (Quantext files or other files).
You can delete files by clicking the
icon.
Running analyses
Click on the 'New' tab in the menu bar, or the
icon (New) on the main dashboard.
The New Analysis panel will open along with the File Manager. Select files to analyse from the File Manager. Give your analysis a name and click run this analysis. You must select one or more student files to run an analysis. You can also choose to select one or more other files to use as reference materials.
Analysis dashboard
When you run a new analysis, you will be taken to the analysis dashboard. Here you can view statistics about your data, and interact with your data to extract insight.
Summary statistics
There are 3 basic counts summarised: words, bigrams and trigrams (collectively called ngrams). For advanced users, you can view a number of extra statistics and measures, such as named entities and noun phrases, as well as a variety of readability indices (TTR, MLTD, SMOG, etc).
